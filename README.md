# Sql-Murder-Mystery
This project walks through my process of solving the SQL Murder Mystery, a detective-style SQL challenge where the goal is to identify the murderer and mastermind
Project Overview
The database contains tables such as crime_scene_report, person, drivers_license, interview, and others.
## How I solved the case
Filtering with WHERE
Pattern matching with LIKE
Inner and left joins
Relating IDs across tables
Narrowing suspects using logical conditions
Following witness clues to identify the killer

## Skills Demonstrated
SQL querying and joining multiple tables
Using logical reasoning for investigative SQL analysis
Debugging issues in query results

## Final Result
After investigating crime scene reports, witness interviews, vehicle records, gym check-ins, and person records, I successfully identified the culprit and the mastermind behind the case.
